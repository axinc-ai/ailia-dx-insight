Setup

Download

Download ailia DX Insight and unzip the zip file.

Installation

For Windows, open ailia_dx_insight.msix and install ailia_dx_insight.exe.
For macOS, open ailia_dx_insight.dmg and install ailia_dx_insight.app into the Application folder.

Launch

For Windows, click on ailia_dx_insight.exe from the Start menu to launch it.
For macOS, right-click on ailia_dx_insight.app and choose open to launch it.

Obtaining OpenAI's API Key

Register for an account on the OpenAI website and obtain OpenAI's API key. You can use ailia DX Insight without entering the OpenAI API key, but the functionality will be limited.

About OpenAI Rate Limitations

If Usage tiers is Free, GPT-3.5 Turbo can be used.
GPT-4 Turbo can also be used by raising Usage tiers to Tier 1.
GPT-3.5 Turbo and GPT-4 Turbo can be used by moving up to Tier 1 after the free trial ends.
To move to Tier 1, you will need to upgrade to a paid account and pay $5.
You can log in to OpenAI's My Page at the link below to check your current usage limits and move to Tier 1.
Settings Limits Page

Setup

A tutorial screen will be displayed at the first launch.
Select the license file that can be downloaded from the trial version email of ailia DX Insight and start.


At the final screen of the tutorial, you can set up the OpenAI API key.

Setting the OpenAI API Key

  • To set it up from the tutorial
    1. Proceed to the end of the tutorial, to "Set API Key".
    2. Enter your OpenAI API key starting with "sk-" in the textbox under "OpenAI API Key".


    3. Press the "Done" button.
  • To set it up from the regular screen
    1. Click the gear icon on the top right corner of the screen.


    2. Enter your OpenAI API key starting with "sk-" in the textbox displayed in the dialog.


    3. Select the version of chatGPT you want to use and press the "Close" button.

Downloading the AI Model

After finishing the initial tutorial, the download of the AI model will start.
The progress of the download can be checked at the bottom of the sidebar on the left.


After the download is complete, ailia DX Insight will be ready for use.
