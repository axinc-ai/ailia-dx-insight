v1.3 release note

Version 1.3 Release Notes

UI Improvements

  • Changed to display a new chat start screen at startup

  • Changed to stop auto-scrolling when scrolling while streaming

  • Settings are now saved when changing chat mode via the query bar

  • Added dark mode

New Features

  • Support for gemma3 through ailia LLM update
  • Added the ability to specify aspect ratio for image generation
  • Added index management feature
  • Added a feature to automatically update the index in the selected folder
  • Added Gemini and gpt-image-1 to LLM selection for image generation

Usability Improvements

  • Changed image generation timeout from 30 seconds to 90 seconds
  • Changed chat history names for image generation and summarization to be more understandable
  • Display detailed error when there is a config file update error
  • Renamed translation results to new translations

Initial Setup Update

  • Changed ailia LLM to be set by default

Terms of Use

  • Display terms of use at startup

Optimization

  • Reduced app startup time

Bug Fixes

  • Fixed an issue where keyboard input would not be accepted on macOS

When Migrating from Version 1.2.1

  • Regarding reading pptx, since the extensions for index targets are saved in the database, if migrating from 1.2.1 you need to add pptx to the index target extensions on the settings screen.
