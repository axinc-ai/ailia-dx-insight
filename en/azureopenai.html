How to connect to Azure OpenAI Service

Overview of Azure OpenAI Service

ailia DX insight utilizes ChatGPT for text generation. In addition to the OpenAI API, it is also possible to use ChatGPT through the Azure OpenAI Service. For more details on the Azure OpenAI Service, please see here.

How to Connect to Azure OpenAI Service

Calling the Settings Screen

On the initial screen of ailia DX insight, click the gear icon in the top right corner to display the settings window.

Displaying the ChatGPT Model Selection Menu

At the top left of the settings menu, there is a ChatGPT model selection menu. Clicking this will display a list of models. Please click the add button located below the ChatGpt model selection menu. Then, a window will be displayed asking whether to add a custom model or Azure OpenAI. Please click 'Azure OpenAI'.

Registering the Configuration of Azure OpenAI Service

A window for registering the ChatGPT deployed on Azure will open. Fill in each field.

  • API Key: The API key of the resource where ChatGPT was deployed.
  • Deployment Name: The deployment name set when deploying ChatGPT.
  • API Version: The version of API being used.
    • The "Supported Versions" page here shows the available versions.
  • Resource Name: The name of the resource where ChatGPT was deployed.
    • This corresponds to the RESOURCE_NAME part in the endpoint "https://RESOURCE_NAME.openai.azure.com/" where you deployed ChatGPT.
  • Model: The name of the deployed model (e.g., gpt-3.5-turbo).
  • Description: Use as notes if necessary.
