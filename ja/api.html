ailia DX Insight 1.0.0
API

Overview

ailia DX Insight は、AI 機能と対話するために使用できる API を公開します。

Specification

Info

  • Path: /info
  • Method: GET
  • Description: Returns the server info
  • Response:
    • Content Type: application/json
    • Schema:
      Key Value Type
      service ailia-insight String
      version Version text String
    • Example:
      {
      "service": " ailia-insight",
      "version": "0.0.0"
      }

Search

  • Path: /search
  • Method: GET
  • Description: Perform a text search
  • Header:
    Key Value Required
    x-axinc-authkey Authentication key No
  • Query:
    Key Value Required
    text Text to search Yes
  • Response:
    • Content Type: application/json
    • Schema:
      Key Value Type
      service ailia-insight String
      version Status code (0 for success) String
      status 0 on success Integer
      text Response text String
    • Example:
      {
      "service": " ailia-insight",
      "version": "0.0.0",
      "status": 0,
      "text": "Response text"
      }
構築:  1.9.3
